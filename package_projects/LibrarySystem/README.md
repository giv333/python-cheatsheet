# 📚 LibrarySystem Package

This is a beginner-friendly Python packaging project using class, inheritance, and module import.

---

## 🚀 What It Includes

- `Book` class: stores title, author, and year  
- `BorrowedBook` class: inherits from Book and adds borrower name  
- A test script (`main.py`) to run the program  
- Folder structure like a real Python package


---

## 📁 Folder Structure
 ```
LibrarySystem/
├── library/
│   ├── __init__.py
│   └── book.py
├── main.py
└── README.md
```
